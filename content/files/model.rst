.. _modelFile:

Model File
==========

For a given physical property (conductivity in this case), the model file has the following format:


|
| :math:`\sigma_1`
| :math:`\sigma_2`
| :math:`\sigma_3`
| :math:`\sigma_4`
| :math:`\sigma_5`
| :math:`\sigma_6`
| :math:`\;\vdots`
| :math:`\sigma_{N-1}`
| :math:`\sigma_N`
|
|

where :math:`N` is the total number of cells in the corresponding tensor or Octree mesh.
